Welcome to First Steps.

If you've got a startup idea, this workshop can help you find out whether it's worth pursuing. Sign up below or read on for a quick description of the curriculum.

Protype

We start prototyping not by building, but by answering three questions:

  • Who's the user I want to serve?
  • What need will I satisfy for them?
  • Will people actually engage with (e.g. buy) the product?

Experiment

Once we have something to show to potential customers, we do just that. We'll discuss several quick tools you can use to get feedback in absolute minimal time. You'll get tips on a few lightweight "testing stacks":

  • Unbounce + Paypal/venmo + Bing
  • Keynote + ipad
  • Shopify + Squarespace

Next Steps

We want your learning to be actionable! At the end of this workshop, we hope you'll find one of three things:

  • The idea is a great success -- time to put in more resources!
  • The idea need to change -- we learned about something that customers would like more
  • This one isn't worth pursuing. On to the next challenge!
